# 《南都爱情故事》

这是基于 Abeto 网页游戏《Messenger》的小世界、轻任务、强氛围结构启发而来的原创叙事项目。

项目核心不是复刻原版，而是以旧南京为舞台，讲述一个年轻邮差送出一封封“有情之书”的慢递故事。

## 核心方向

- 旧南京
- 慢递
- 情书 / 有情之书
- 时间选择
- 喜悲交接
- 低压力探索
- 从前慢式浪漫

## 与原版 Messenger 的关系

原版 Messenger 的静态快照仍保留在仓库根目录，用于学习它的小地图结构、网页运行方式、氛围节奏和资源组织方式。

《南都爱情故事》应使用原创内容、原创或授权资产，不直接使用原版 Messenger 的美术、音频、模型或压缩资源作为正式发布内容。
